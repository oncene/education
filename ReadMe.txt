Installation Document

Node version: 11.11.0
npm version: 6.12.0

After getting the project from git, Goto the project_folder/Web


**Note: Use [yarn](https://yarnpkg.com/en/docs/install) instead npm**

Installing all the dependencies of project in project directory, run following command:
 ` yarn `


Then on local system, run command : 
` yarn start `
